### 🚀 Featured Projects

**[Vela Cloud — Knowledge-Base Assistant (RAG)](https://github.com/harshalbalar/ask-vela-RAG-assistant)**
Production-style RAG pipeline with hybrid search (BM25 + semantic), cross-encoder re-ranking, and source-cited answers with honest abstention. Re-ranking raised MRR to **0.86 (+41%)** and Hit@4 to **0.96**; a free local embedder matched a paid SOTA API.
`Python` `LangChain` `FastAPI` `Chroma` `Gemini` `Docker`

**[DocMind AI — Chat with Your Documents (RAG)](https://github.com/harshalbalar/docmind-ai)**
Multi-format document Q&A (PDF, DOCX, TXT, CSV, MD) with conversational memory, citations, and chat export. History-aware query rephrasing for accurate follow-ups.
`Python` `LangChain` `Streamlit` `FAISS` `Gemini`

**[TurtleBot Card Detection](https://github.com/harshalbalar/Turtlebot-card-detection)**
Real-time computer-vision pipeline running on a TurtleBot to detect and classify cards during live operation, integrated with the robot's ROS stack.
`Python` `OpenCV` `ROS`
